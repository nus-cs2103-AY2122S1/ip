	Hello! I'm Duke, your personal CLI bot! :D
	I function as a simple TodoList.
	I can keep track of 3 types of tasks:
		> Todo: To add a new todo task, enter 'todo' followed by a task description.
		> Deadline: To add a new deadline, enter 'deadline' followed by the task description and specify the due date using '/by <due-date>'
		> Event: To add a new event, enter 'event' followed by the event description and specify the time using '/by <time-stamp>'
	- To see all your tasks, enter 'list'.
	- To delete a task, enter 'delete' followed by the index of the task you wish to delete (e.g delete 2).
	- You can also mark tasks as done by typing 'done' followed by the index of the task you completed (e.g done 2).
	- Once you are done, just enter 'bye' to quit the program.
	Have fun! ^_^
